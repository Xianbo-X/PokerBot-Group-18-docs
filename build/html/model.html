model module¶

model.build_model(model_name: str)¶

Prepare the model.

Parameters

model_name (str) – The definition of model. It should be a module that model class are defined. Predefined model_names are “models.CNN3”, “models.CNN4”

Returns

model – Model definition (untrained).

Return type

model class from any toolbox you choose to use.

Examples

>>> build_model("models.CNN3")
    Return models which is defined in the python file CNN3.py under package models
>>> build_model("models.CNN4")
    Return models which is defined in the python file CNN4.py under package models

Parameters

  • filename (str, default: "model.pt") – The file name of the saved model.
